## ‚ùì Frequently Asked Questions

### General Questions

**Q: What is PUFF?**
> A: PUFF is a free and secure skin changer application developed for League of Legends. It allows you to change the appearance of champions in-game.

**Q: Is PUFF legal? Will I get banned?**
> A: PUFF only makes visual changes and doesn't permanently interfere with game files. However, like all 3rd party software, use is at your own responsibility. We recommend reading Riot Games' terms of use.

**Q: Is it paid?**
> A: PUFF is completely free.

**Q: Which platforms does it work on?**
> A: Currently only Windows 10/11 (64-bit) is supported.

**Q: Is there a version for Mac or Linux?**
> A: Not yet, but it's planned for the future.

### Installation and Running

**Q: How do I install?**
> A: No installation required! Download the `.exe` file and double-click. It's a portable application.

**Q: Windows Defender gives a warning?**
> A: This is normal for unsigned applications. Use "More info" ‚Üí "Run anyway" option. The application is completely safe.

**Q: Can't find League of Legends path?**
> A: Manually select the `LeagueClient.exe` file from Settings ‚Üí League Path ‚Üí Browse menu. It's usually in `C:\Riot Games\League of Legends\`.

**Q: I get an error when running?**
> A:
> - Update Windows
> - Install .NET Framework 4.7.2
> - Install Visual C++ Redistributable
> - Try running as administrator

### Usage Questions

**Q: How are skins applied?**
> A: Select champion, click on your desired skin, and press "Apply" button. It's that simple!

**Q: Does only I see it?**
> A: Yes, skin changes are only visible on your computer. Other players see your default skin.

**Q: Can I apply skin for multiple champions?**
> A: Yes! You can apply desired skins for all champions.

**Q: How do I remove skin?**
> A: Click "Clear" button or restart the game.

**Q: How do I add custom .fantome skins?**
> A: Custom Skins ‚Üí Import Custom Skin ‚Üí Select `.fantome` file ‚Üí Fill metadata ‚Üí Save.

### Automation Questions

**Q: Does Auto Accept always work?**
> A: Yes, when enabled, PUFF runs in background and auto-accepts when match is found.

**Q: Is Auto Ban/Pick safe?**
> A: Yes, but you can always intervene manually. You can add delays and security measures from settings.

**Q: Auto skin not applying?**
> A:
> - Make sure Auto Skin feature is on
> - Make sure you've set favorite skin for champion
> - Check if PUFF is running in background

### Technical Questions

**Q: How much RAM does PUFF use?**
> A: Usually between 150-300 MB, varies depending on settings.

**Q: Is internet connection required?**
> A: Yes for initial setup and updates. Can work offline in normal use.

**Q: Where is my data stored?**
> A: In `%APPDATA%/copief/` folder, encrypted.

**Q: How do I backup my settings?**
> A: Manually copy `%APPDATA%/copief/` folder.

**Q: How much space do skins take?**
> A: Between 100-500 MB depending on cache size. Can be cleaned from settings.

### Troubleshooting

**Q: PUFF won't open?**
> A:
> 1. Run as administrator
> 2. Temporarily disable antivirus
> 3. Add exception to Windows Defender
> 4. Re-download the application

**Q: Skin not applying?**
> A:
> 1. Make sure League of Legends is closed
> 2. Click "Repair" button
> 3. Restart PUFF
> 4. Run with administrator permission

**Q: Can't update?**
> A:
> 1. Check your internet connection
> 2. Check firewall settings
> 3. Manually download new version
> 4. Delete old version and install new one

**Q: Can't import custom skin?**
> A:
> 1. Make sure file has `.fantome` extension
> 2. Check if file is not corrupt
> 3. Try again from different location
> 4. Restart PUFF
